# doTest

This application is built using HTML, CSS, JavaScript and Firebase. It provides an online test platform where users can login, select subjects, attempt tests, view scores, review answers and track their test history.

This is a modern exam practice project.

## Technologies Used In This Application

1. HTML
2. CSS
3. JavaScript
4. Firebase Authentication
5. Firebase Firestore
6. Font Awesome Icons
7. Google Fonts

## Features

1. Google login authentication
2. Subject wise test selection
3. Multiple choice question test system
4. Timer based exam interface
5. Score calculation and result modal
6. Answer review after test completion
7. User dashboard
8. Test history tracking
9. Analytics and leaderboard pages
10. Printable report page

## How To Run This Project

1. Clone or download this project.
2. Open the project folder.
3. Run the project using a local server.
4. Open `index.html` in the browser.
5. Login with Google and start the test.
